# Agent Notes

## Build
- Build from the repository root with:
  - `dotnet build`

## Solution
- Solution file: `TLDAccessibility.sln` (repository root).

## Build output (mod DLL)
- The mod DLL is produced under the Code project output folder:
  - `Code/bin/Debug/TLDAccessibility.dll` (Debug)
  - `Code/bin/Release/TLDAccessibility.dll` (Release)

## MelonLoader logs
- Logs are written under the game install directory, typically:
  - `<TLD install>/MelonLoader/Latest.log`
  - `<TLD install>/MelonLoader/Logs/`
